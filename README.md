# linuxweb
linux web控制面板

一个用python做的linux web面板,可以通web来管理linux

现在主要的功能是搭建高性能的web网站  
<br>
<br>
<br>
<br>
  
  


运行方式

1.pip install -r requirements.txt

2.python runserver.py

<br>
<br>
<br>
<br>

如果你不想运行web界面,只是想安装高性能php环境,直接执行 cd project/shell/auto &&　sh autoinstall.sh
